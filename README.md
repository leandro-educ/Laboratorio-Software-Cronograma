# Campus · Laboratorio de Software

Aplicación web estática desarrollada con **HTML + CSS + JavaScript** para presentar la materia **Laboratorio de Software** como un campus navegable por semanas.

## Contenido del proyecto

- `index.html` — portada principal del campus
- `herramientas.html` — entorno de trabajo, stack y comandos base
- `styles.css` — estilos generales del campus
- `script.js` — interacciones básicas (tema, reveal, progreso, año automático)
- `laboratorio-software.png` — imagen principal de portada
- `paginas/semana01.html` a `paginas/semana16.html` — recorrido semanal

## Base de contenido

El cronograma y la organización semanal fueron adaptados a partir del archivo:

- `Cronograma_y_plan_de_clases-Final.xlsx`

## Cómo usar

1. Descomprimir el proyecto.
2. Abrir `index.html` en el navegador.
3. Navegar por las semanas desde la portada o desde cada página semanal.

## Características

- Diseño visual inspirado en el campus de referencia.
- Navegación por 16 semanas.
- Modo claro / oscuro.
- Sección de herramientas.
- Páginas semanales con:
  - tema
  - objetivo pedagógico
  - foco técnico
  - actividades esperadas
  - continuidad de navegación
